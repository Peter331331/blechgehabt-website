
# Blech gehabt Website

## 📦 Inhalt
- index.html: Startseite im Stil von buena banda
- style.css: Dunkles, responsives Layout
- bilder/: Platzhalterbilder für Header und Galerie

## 🚀 GitHub Pages Anleitung
1. GitHub-Account erstellen unter https://github.com
2. Neues Repository anlegen (z. B. blechgehabt-website)
3. Dateien aus diesem ZIP hochladen
4. In den Repository-Einstellungen unter „Pages“:
   - Branch: `main`
   - Ordner: `/ (root)`
5. Fertig! Deine Seite ist live unter `https://DEINNAME.github.io/blechgehabt-website`

## 🖼️ Eigene Bilder einfügen
- Lege deine Bilder im Ordner `bilder/` ab
- Ersetze die vorhandenen Dateien oder passe die Dateinamen im HTML an

Beispiel:
```html
<img src="bilder/deinbild.jpg" alt="Beschreibung">
```

Viel Spaß!
